# 日记便签 (DayMemo) - 思源笔记 Memos 风格插件

一个轻量级的、受 [Memos](https://github.com/usememos/memos) 启发的快速笔记插件，适用于[思源笔记](https://b3log.org/siyuan)。在思源的主内容区中快速记录想法、添加标签、搜索和过滤。

![SiYuan](https://img.shields.io/badge/思源笔记-≥3.3.0-blue) ![License](https://img.shields.io/badge/许可证-MIT-green)

## 功能特性

- **主标签页视图** — 在思源的主内容区打开，采用 Memos 风格的双栏布局
- **侧栏 Dock 面板** — 同时支持在思源侧栏（左/右/底部）显示为紧凑的单栏面板，无需离开当前标签页即可快速记录
- **Markdown 支持** — 粗体、斜体、代码、链接、图片等，内联渲染
- **交互式清单** — 使用 `- [ ]` 和 `- [x]` 语法创建任务列表，复选框可点击，状态自动保存
- **Mermaid 图表** — 使用 ` ```mermaid ` 围栏代码块渲染流程图、时序图等，利用思源内置的 Mermaid 引擎，无需额外依赖
- **图片与附件上传** — 支持上传图片（按钮、粘贴、拖拽）和任意文件附件（zip、pdf 等），存储在思源 `data/assets/` 目录下，自动随云端同步
- **多级标签系统** — 使用 `#标签` 或 `#父级/子级/孙级` 语法（类似 Flomo）组织便签；侧栏以可折叠树形结构展示标签，支持展开/收起；点击父级标签自动筛选其所有子标签
- **日历视图** — 真实月历，支持 `‹` `›` 翻月，「今天」按钮快速回到当月，按便签密度显示热力图着色，点击日期筛选
- **时间线视图** — 便签按日期分组，最新的在前
- **搜索** — 在侧栏中全文搜索所有便签
- **过滤标签** — 在 全部 / 已置顶 / 已归档 视图间切换
- **添加到今日笔记** — 一键将便签内容追加到该便签创建日期对应的思源「今日笔记」文档中，自动标注来源；支持自定义路径模板和丰富的日期变量（详见下方设置说明）
- **正则替换规则** — 在设置中定义自定义的正则查找替换规则，在将便签添加到今日笔记之前自动应用 — 例如将 `#任务 ` 开头的行转换为 `- [ ] ` 待办清单项
- **右键菜单** — 右键点击便签可快速操作：编辑、置顶、归档、添加到今日笔记、设置提醒、复制内容、删除
- **提醒功能** — 通过右键菜单为任意便签设置提醒，时间选择器默认为当前时间后 10 分钟；到达提醒时间时，同时弹出思源应用内通知和浏览器系统通知
- **标注功能** — 对任意便签添加标注，创建关联笔记（类似 Flomo 的批注功能）；标注与源便签通过双向链接连接，方便互相跳转查阅；标注预览直接显示在源便签下方；删除便签时自动清理所有双向标注链接，保持数据一致性
- **随机回顾** — 侧栏底部一键「随机回顾」，以卡片形式随机展示便签；每次显示 5 条，支持前后翻阅和换一批 — 灵感来自 Flomo 的每日回顾
- **置顶和归档** — 将重要便签置顶，归档旧便签减少干扰
- **云同步安全** — 基于时间戳的合并逻辑，配合软删除墓碑机制，安全支持思源云端多设备同步
- **深色模式** — 自动跟随思源主题
- **多语言** — 支持中文和英文

## 使用方法

1. 点击顶部工具栏的 **日记便签** 图标打开完整标签页视图，或在侧栏 Dock 中找到 **日记便签** 使用紧凑面板
2. 在编辑区输入便签内容，使用 `#标签` 或 `#父级/子级` 添加多级标签
3. 添加图片（点击 🖼、从剪贴板粘贴或拖拽）和文件附件（点击 📎）— 自动上传至思源 `assets/` 目录
4. 使用 `- [ ]` 创建可点击的清单，使用 ` ```mermaid ` 代码块绘制图表
5. 按 `Ctrl+Enter` 或点击保存
6. 使用过滤标签页（全部 / 已置顶 / 已归档）切换视图
7. 点击日历上的日期筛选该天的便签；使用 **今天** 按钮快速回到当月
8. 点击侧栏中的标签按钮按标签过滤
9. 鼠标悬停在便签上可以看到编辑 / 置顶 / 归档 / 标注 / 📅 添加到今日笔记 / 删除操作，也可以**右键点击**便签打开上下文菜单（含**设置提醒**和**标注**）
10. 点击便签上的标注按钮即可创建标注 — 一条与源便签关联的新便签；标注预览显示在源便签下方，点击可在标注与源便签间快速跳转
11. 双击便签内容可快速进入编辑模式
12. 可直接选中并复制便签中的文本内容，也可通过右键菜单复制完整便签内容

## 设置

打开插件设置（在思源的 集市 → 已安装 中找到 DayMemo 插件卡片，点击齿轮图标）：

- **Daily Note 路径模板** — 自定义「添加到今日笔记」功能使用的路径模板。留空则使用笔记本默认的 `dailyNoteSavePath`。示例：

  ```
  /Daily Note/{{now | date "2006/01"}}/第{{now | ISOWeek}}周/{{now | date "2006-01-02"}}-周{{now | WeekdayCN}}
  ```

  支持的模板变量：

  | 模板 | 说明 | 输出示例 |
  |------|------|---------|
  | `{{now \| date "2006-01-02"}}` | Go 风格日期格式 | `2026-04-07` |
  | `{{now \| date "15:04:05"}}` | Go 风格时间格式 | `14:30:00` |
  | `{{now \| ISOWeek}}` | ISO 周数 | `15` |
  | `{{now \| ISOYear}}` | ISO 周年份 | `2026` |
  | `{{now \| Weekday}}` | 星期几（0=周日） | `2` |
  | `{{now \| WeekdayCN}}` | 中文星期 | `二` |
  | `{{now \| WeekdayCN2}}` | 带前缀中文星期 | `周二` |

  Go 日期格式占位符：`2006`（年）、`01`（月）、`02`（日）、`15`（24小时制时）、`03`（12小时制时）、`04`（分）、`05`（秒）、`PM`/`pm`、`Monday`/`Mon`、`January`/`Jan`。

- **添加到今日笔记 - 使用当前日期** — 开启后，添加到日记时使用当前系统日期；关闭（默认）则使用便签的创建日期。

- **启用正则替换** — 开关控制是否在添加便签到今日笔记时启用正则替换规则。

- **替换规则** — 按顺序应用于便签内容的正则查找替换规则列表。每条规则包含：
  - **匹配模式** — JavaScript 兼容的正则表达式（使用 `gm` 标志）。示例：`^#任务 `
  - **替换文本** — 替换后的文本。示例：`- [ ] `

  使用 `+` / `-` 按钮添加或删除规则。匹配模式为空的规则会被忽略。

## 数据存储与同步

- 便签数据：`data/storage/petal/siyuan-plugin-day-memo/memos-data`
- 插件设置：`data/storage/petal/siyuan-plugin-day-memo/settings`

以上文件均会自动纳入思源的云端同步。

上传的图片和附件存储在思源标准的 `data/assets/` 目录下，同样会纳入云端同步。

**多设备安全**：加载数据时，插件会执行基于时间戳的合并 — 对每条便签，保留 `updatedAt` 更新的版本。删除使用软删除（墓碑）标记，确保删除操作在多设备间正确传播。

## 开发

```bash
# 安装依赖
pnpm i

# 开发构建（监听模式）
pnpm dev

# 生产构建（生成 package.zip）
pnpm build
```

## 技术栈

- **语言**：TypeScript（原生 DOM，无框架）
- **构建**：Webpack + esbuild-loader + SCSS
- **运行时**：思源插件 API（petal v1.1.7）
- **最低思源版本**：3.3.0

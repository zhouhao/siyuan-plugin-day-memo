# 日记便签 (DayMemo) - 思源笔记 Memos 风格插件

一个轻量级的、受 Memos 启发的快速笔记插件，适用于[思源笔记](https://b3log.org/siyuan)。在思源中快速记录想法、添加标签、搜索和过滤。

## 功能特性

- **快速记录** — 通过侧栏面板或快捷键 (`Alt+Cmd+N`) 即时记录想法
- **Markdown 支持** — 粗体、斜体、代码、链接等
- **标签系统** — 使用 `#标签` 语法组织便签，自动提取和过滤
- **时间线视图** — 便签按日期分组，最新的在前
- **搜索** — 全文搜索所有便签
- **置顶和归档** — 将重要便签置顶，归档旧便签
- **深色模式** — 自动跟随思源主题
- **多语言** — 支持中文和英文

## 使用方法

1. 从右侧栏打开 **日记便签** 面板
2. 在编辑区输入便签内容，使用 `#标签` 添加标签
3. 按 `Ctrl+Enter` 或点击保存
4. 使用过滤标签页（全部/已置顶/已归档）和搜索栏查找便签
5. 点击侧栏中的标签按钮按标签过滤
6. 鼠标悬停在便签上可以看到编辑/置顶/归档/删除操作

## 快捷键

| 快捷键 | 操作 |
|--------|------|
| `Alt+Cmd+M` | 切换日记便签面板 |
| `Alt+Cmd+N` | 快速记录对话框 |
| `Ctrl+Enter` | 保存便签（在编辑器中） |
